## Core Modules

- Dashboard analytics and daily summaries
- Worker management
- Bar operations: drinks, depot inventory, issuance, stocktaking, daily closing
- Restaurant operations: tracked items, kitchen issuance, inventory, daily closing
- Hotel operations: rooms, bookings, daily closing
- Archive and monthly summaries
- AI Assistant for business Q and A
- Desktop app download and installer support
